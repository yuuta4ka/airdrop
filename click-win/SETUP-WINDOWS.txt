AIRDROP ON WINDOWS - HOW TO GET FILES WITHOUT BREAKING THEM
==========================================================

PROBLEM: ZIP / Telegram / USB can break .bat files (encoding, line endings).
         You may see errors like: 'ho' is not recognized, 'agnose.bat' ...

BEST WAY: git clone (fast, correct files)

  1. Install Git: https://git-scm.com/download/win
  2. Install Node.js: https://nodejs.org (then REBOOT PC)
  3. Open cmd and run:

     cd %USERPROFILE%\Desktop
     git clone https://github.com/yuuta4ka/airdrop.git
     cd airdrop
     click-win\Start.bat

  4. Open http://localhost:8080

UPDATE PROJECT (after changes on Mac):

     cd path\to\airdrop
     click-win\Update-Project.bat
     click-win\Start.bat

  No npm install needed. Server packages are bundled in vendor\ folder.

DO NOT:
  - Send .bat files through Telegram (they get corrupted)
  - Open site as file:///.../index.html (need Start.bat running)
  - Run npm install for normal site use (only needed for Vite dev on Mac)

IF YOU MUST USE USB:
  - Copy the whole airdrop folder (not only click-win)
  - After copy, run Diagnose.bat first

TROUBLESHOOT:

  Server won't start:
     click-win\Diagnose.bat
     git pull
     click-win\Start.bat

  Old broken node_modules from previous versions:
     rmdir /s /q node_modules
     click-win\Start.bat
     (Start creates fresh links from vendor\ - no npm needed)
